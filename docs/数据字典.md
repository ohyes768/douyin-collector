## 1. 数据模型

### 1.1 VideoInfo - 视频信息

**说明**：从抖音采集的视频元数据

**数据结构**

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| aweme_id | string | 是 | 抖音视频唯一标识 | "7123456789012345678" |
| title | string | 是 | 视频标题 | "视频标题" |
| author | string | 是 | 作者昵称 | "作者名称" |
| video_url | string | 是 | 视频下载地址 | "https://v11-weba.douyinvod.com/..." |
| desc | string | 否 | 视频描述 | "详细描述内容" |
| create_time | int | 否 | 创建时间戳 | 1769258264 |
| is_product | bool | 否 | 是否为商品视频 | false |

**Python 定义**

```python
@dataclass
class VideoInfo:
    aweme_id: str
    title: str
    author: str
    video_url: str
    desc: str = ""
    create_time: int = 0
    is_product: bool = False
```

## 2. 配置模型

### 2.1 应用配置 (app.yaml)

**文件位置**：`config/app.yaml`

**配置结构**

```yaml
app:
  server: { ... }      # 服务器配置
  cookie: { ... }      # Cookie 配置
  collector: { ... }   # 采集配置
  download: { ... }    # 下载配置
  processing: { ... }  # 处理配置
  logging: { ... }     # 日志配置
```

### 2.2 服务器配置 (server)

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| url | string | 是 | - | file-system-go 服务器地址 |
| timeout | int | 否 | 60 | 请求超时时间（秒） |
| check_endpoint | string | 否 | "/api/check" | 文件检查接口路径 |
| upload_endpoint | string | 否 | "/upload" | 文件上传接口路径 |

**示例**

```yaml
server:
  url: "http://your-ecs-ip:8000"
  timeout: 60
  check_endpoint: "/api/check"
  upload_endpoint: "/upload"
```

### 2.3 Cookie 配置 (cookie)

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| path | string | 是 | "config/cookie.yaml" | Cookie 配置文件路径 |

### 2.4 采集配置 (collector)

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| days_limit | int | 否 | 7 | 采集最近 N 天的视频 |
| days_start | int | 否 | 0 | 过滤 N 天前的视频（0 表示不限制） |
| max_videos | int | 否 | 0 | 最大采集数量，0 表示不限制 |
| max_file_size | int | 否 | 104857600 | 最大文件大小（字节），默认 100MB |
| exclude_products | bool | 否 | true | 是否排除商品视频 |

**时间范围过滤说明**

- `days_limit`：采集最近 N 天的视频（如 7 表示最近 7 天）
- `days_start`：过滤 N 天前的视频（如 3 表示过滤 3 天前的视频）
- 组合使用：`days_limit=7, days_start=3` 表示获取 3-7 天前收藏的视频
- 单独使用：`days_limit=7, days_start=0` 表示获取最近 7 天的视频

### 2.5 下载配置 (download)

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| cache_dir | string | 否 | "cache/videos" | 视频缓存目录 |
| chunk_size | int | 否 | 1048576 | 下载分块大小（字节），默认 1MB |
| timeout | int | 否 | 300 | 下载超时时间（秒） |

### 2.6 处理配置 (processing)

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| max_retries | int | 否 | 3 | 失败重试次数 |
| retry_delay | int | 否 | 5 | 重试延迟时间（秒） |
| serial | bool | 否 | true | 是否串行处理 |

### 2.7 日志配置 (logging)

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| level | string | 否 | "INFO" | 日志级别 |
| console | bool | 否 | true | 是否输出到控制台 |
| file | bool | 否 | true | 是否输出到文件 |
| dir | string | 否 | "logs" | 日志目录 |

## 3. Cookie 配置 (cookie.yaml)

**文件位置**：`config/cookie.yaml`

**配置结构**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| cookie | string | 是 | 抖音 Cookie 字符串 |
| timeout | int | 否 | 请求超时时间（秒） |
| max_retries | int | 否 | 重试次数 |

**示例**

```yaml
douyin:
  cookie: "sessionid=xxx; ttwid=xxx; ..."
  timeout: 30
  max_retries: 3
```

## 4. 处理结果模型

### 4.1 处理结果 (ProcessResult)

**说明**：单个视频的处理结果

| 字段 | 类型 | 说明 |
|------|------|------|
| aweme_id | string | 视频 ID |
| title | string | 视频标题 |
| success | bool | 是否成功 |
| skipped | bool | 是否跳过（已存在） |
| error | string | 错误信息 |

### 4.2 处理统计 (ProcessStats)

**说明**：批量处理的统计信息

| 字段 | 类型 | 说明 |
|------|------|------|
| total | int | 总数 |
| success | int | 成功数 |
| skipped | int | 跳过数 |
| failed | int | 失败数 |

## 6. 取消收藏数据模型

### 6.1 取消收藏结果 (UncollectResult)

**说明**：单个视频取消收藏的结果

| 字段 | 类型 | 说明 |
|------|------|------|
| aweme_id | string | 视频 ID |
| success | bool | 是否成功取消收藏 |
| error | string | 错误信息（失败时） |
| screenshot_path | string | 调试截图路径（可选） |

### 6.2 批量取消收藏统计 (UncollectStats)

**说明**：批量取消收藏的统计信息

| 字段 | 类型 | 说明 |
|------|------|------|
| total | int | 总数 |
| success | int | 成功数 |
| failed | int | 失败数 |
| success_rate | float | 成功率 |
